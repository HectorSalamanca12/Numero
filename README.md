# 🎲 Número Secreto - Jogo Interativo

## 📌 Sobre o Projeto  
**Número Secreto**  é um jogo divertido, com o objetivo de adivinhar um nuemro secreto, e a cada tentativa do usuario o programa da dicas se o numero é **Maior** ou **Menor**, será disponibilizado 3 chances e ao final uma chuva de fogos toma a tela.





---


🔗 **Demonstração:** [Clique para jogar](https://hectorsalamanca12.github.io/numero/)

---

## 🎯 Funcionalidades  
- ✅ Geração automática de número secreto aleatório  
- ✅ Validação das tentativas com dicas personalizadas  
- ✅ Interface leve e responsiva  
-

## 🛠️ Tecnologias Utilizadas  
- **HTML5** – Estrutura da página  
- **CSS3** – Estilização do layout

```
/numero_secreto
│── /css/           → Estilos personalizados (se houver)
│── /js/
│   └── script.js   → Lógica do jogo
└── index.html      → Página principal do jogo
```

---

## 🚀 Como Jogar  
1. Abra o arquivo `index.html` em qualquer navegador.  
2. O jogo irá gerar um número secreto aleatório.  
3. Digite um número no campo de entrada e clique em **Enviar**.  
4. O jogo indicará se o número é **maior** ou **menor** que o segredo.  
5. Continue tentando até acertar!
